# 🎮 Day 1 - Number Guessing Game

## 📖 Description

This is a simple Python Number Guessing Game built as my first Python project.

The computer randomly selects a number between **1 and 100**, and the player has to guess it. After each guess, the program provides a hint indicating whether the guess is **too high** or **too low**. The game continues until the correct number is guessed, and it displays the total number of attempts taken.

---

## 🎯 Features

* Random number generation
* User input
* Too High / Too Low hints
* Unlimited attempts
* Displays the total number of guesses
* Beginner-friendly console application

## ▶️ How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the program:

```bash
python number_guessing_game.py
```

4. Enter your guesses until you find the correct number.
