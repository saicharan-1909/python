# Day 2 - Word Counter

## Description
A Python program that reads a text file and counts how many times each word appears.

## Concepts Used
- File Handling
- Dictionaries
- Loops
- Strings

## Sample Input

python is fun
python is easy
fun to learn python

## Sample Output

python : 3
is : 2
fun : 2
easy : 1
to : 1
learn : 1
